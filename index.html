Mi trabajos

Lista de tareas

El proyecto de Lista de Tareas es una aplicación web que permite a los usuarios gestionar sus tareas diarias de manera eficiente. Con esta aplicación, los usuarios pueden agregar nuevas tareas, editar las existentes, marcarlas como completadas y eliminarlas cuando sea necesario.

App para Clima

Clima es una aplicación web que proporciona a los usuarios información en tiempo real sobre las condiciones climáticas. La aplicación utiliza una API de clima para obtener datos actualizados y mostrarlos de manera fácil de entender.

Tienda en Línea

La Tienda en Línea, es una plataforma web diseñada para proporcionar a los usuarios una experiencia de compra intuitiva y eficiente. Ofrece una interfaz atractiva donde los clientes pueden explorar productos, agregarlos al carrito, ajustar cantidades y realizar compras de manera sencilla. Una característica única de esta tienda es que, al confirmar la compra, se redirige automáticamente al cliente al WhatsApp del propietario para facilitar la comunicación directa.
